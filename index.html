敏感与边界感

发表于 2026-02-26 更新于 2026-02-27 分类于 个人状态

在玩MMORPG的多人网游时，组队招募系统弹出一条申请提示，“xxx请求加入你的队伍”，当对方不符合要求时，“拒绝” 和 “忽略”两个选项我总是会选“忽略”，因为觉得选“拒绝”对方会难过，与之对应的是，当别人拒绝我的时候，我也会很难过， 时常觉得自己内心太敏感，玻璃心，这个世界大家好像都不在意这些事情。

在意他人感受，避免让别人不舒服，害怕被拒绝的同时拒绝别人也会让自己不舒服

过度共情+自我价值感敏感+缺乏安全感。这可能与我学生时期遭受过很长一段时间的校园霸凌有关。 我渴望“温柔的世界”，但很明显现实并不是这样，有的时候很难分清是不是我自己的问题，我把游戏里的组队拒绝理解成对人的否定，而不是对条件的否定。但是现实中有很多场景下我都会有相似的感受存在。

我希望自己能活的更加舒服一点。

阿德勒的哲学课

发表于 2025-11-12 更新于 2026-02-27 分类于 读书手记

自卑 / 比较 / 羡慕他人

健全的比较对象永远只能是“理想中的自己”，而非他人，我们不断的前进不是为了和谁竞争，而是为了超越自我。一旦和别人产生竞争关系，就会有赢家和输家，内心永无宁日。

人生就像一个“平台”，我们走在上面，尽管我们之间的速度和距离各不相同，但我们都平等的走在一个平面上，只要自己不断的前进就可以了。

分歧 / 争论

当你和别人意见不合，产生争论时，就会产生 权利之争，对方这个时候已经不再去考虑事情本身的对错，而是想通过 权利之争 让不顺眼的你屈服。不要上当，不要陷进“权利之争”，当受到对方的言语挑衅时，你要做的就是对对方的行为不做任何反应，即使你知道自己是正确的，也不要去责难别人，因为即使你在权力之争中赢了，对方也会怀有一颗复仇之心。

所以绕来绕去还是绕回到那个话题：“不要产生竞争，只要有竞争，就会有赢家和输家”。
